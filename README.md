# О себе

Привет! Меня зовут Артём, я QA-Инженер.

Я выбрал профессию QA‑инженера, потому что мне нравится находить слабые места в продуктах и помогать делать их лучше. Меня мотивирует возможность влиять на качество конечного продукта и обеспечивать лучший пользовательский опыт. Я получаю удовольствие от процесса поиска багов, анализа причин их возникновения и взаимодействия с разработчиками для их устранения.

За время работы я накопил опыт в:
* ручном тестировании веб‑ и мобильных приложений;
* составлении тест‑кейсов, чек‑листов и тест‑планов;
* документировании дефектов и отслеживании их исправления (работа с баг‑трекерами);
* проведении регрессионного, дымового и функционального тестирования;
* подготовке отчётности по результатам тестирования.

Этот опыт позволит мне быстро влиться в новую команду и эффективно выполнять задачи по обеспечению качества продукта.
